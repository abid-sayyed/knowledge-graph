In January 2022, Google Inc. announced a partnership with OpenAI to improve AI infrastructure.

Sundar Pichai, the CEO of Google, stated that the collaboration would focus on cloud scalability and machine learning optimization.

The project included development of a new system called CloudVision Pro, which improves distributed inference performance.

Later in March 2022, Microsoft Corporation joined the initiative, represented by Satya Nadella.

The companies planned a rollout for June 2023.

CloudVision Pro includes the following features

 automatic workload scaling
 GPU resource balancing
 predictive latency reduction
 real-time monitoring dashboard

OpenAI engineers, including Sam Altman, attended the technical review sessions.
